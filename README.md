# React-Crypto-App

## About

Live cryptocurrency tracker with price charts and market data. Built to learn React, charting libraries, and working with financial APIs.

## Tech Stack

React, CoinGecko API, Chart.js

## Getting Started

```bash
git clone https://github.com/shashhii/React-Crypto-App.git
cd React-Crypto-App
```

- Clean and intuitive UI
- Fully functional backend
